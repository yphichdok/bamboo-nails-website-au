## Setup

1. Clone the repository:
```bash
git clone https://github.com/yphichdok/bamboo-nails-website-au.git
```

2. Navigate to the project directory:
```bash
cd bamboo-nails-website-au
```

3. Open `index.html` in a web browser or use a local server:
```bash
# Using Python
python3 -m http.server 3000

# Using Node.js (if you have http-server installed)
npx http-server -p 3000
```

4. Visit `http://localhost:3000` in your browser

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #66BB6A;
    --secondary-color: #4CAF50;
    --gold-luxury: #D4AF37;
    /* ... */
}
```

### Content
- Update service prices in `index.html` and `services.html`
- Replace images in `assets/images/` folders
- Update contact information in `contact.html` and location pages
- Modify social media links in footer sections
